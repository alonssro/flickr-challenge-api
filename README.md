## Description

Flickr Backend hecho en Framework [NestJS](https://docs.nestjs.com/)

## Setup

```bash
$ git clone https://github.com/alonssro/flickr-challenge-api.git
$ cd flickr-challenge-api
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Docs

```bash
$ npm run compodoc
```

Abre tu navegador y navega a http://127.0.0.1:8080
